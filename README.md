# Knowledge Assistant Agent

Using OpenOnion to create a simple AI Agent that scans through a provided pdf file and is capable for answering questions regarding the file
Currently this has some university subject pdfs attached but feel free to attach your own pdfs and give it a go.

## For personal usage
- This does use Gemini-2.5-Flash, so you will need to create a file .env which contains your key in the format: GOOGLE_API_KEY=yourkeyhere
- For installation of the newest version of OpenOnion
